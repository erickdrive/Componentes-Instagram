# Componentes Página Feed Instagram 👋

## Objetivo
```text
Construir uma página feed do Instagram utilizando a componentização através da tecnologia React-Native.
```

## Para Iniciar

1. Navegue até o diretório 

   ```bash
   cd Componentes-Instagram
   ```

2. Instalar as dependências

   ```bash
   npm install
   ```

3. Iniciar o app

   ```bash
   npm run web
   ```
